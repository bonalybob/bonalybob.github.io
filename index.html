Munro

Created as part of my Advanced Higher Computing project, Munro is a Single Page Progressive Web App with offline support made using VueJS with a Restful Python Flask API. Munro calculates and displays the results for a league of Orienteering events, with options to search and sort the results and flexible results upload to reduce volunteer work

Nevis

Simple, Easy to Use Event Management Software for Orienteering created using Electron and VueJS. Read the raw data from the timing chips through the serial port and then calculates the times and results based off the data imported, ready for export in a variety of formats. Includes advanced course matching, and live search and sort of entries

Maze Checker

Originally created for World Orienteering Day in 2018, it is designed to read the timing chips and check a simple maze orienteering course is completed correctly, keeping a simple scoreboard with the winners time. Created using Electron with a simple interface to be as easy to use as possible
